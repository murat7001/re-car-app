## Getting Started

First, download the necessary things:
```bash
npm install
```
```bash
npm install -g json-server
```
Run the development server:
```bash
npm run dev
```
```bash
npm run json
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
